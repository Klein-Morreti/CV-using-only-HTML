# CV using only HTML

This is my solution to the [Single-Page HTML CV Project](https://roadmap.sh/projects/single-page-cv) on roadmap.sh.

## How to Run
- If you have **Live Server** in VS Code, you can open it directly with Live Server.
- If not, right-click the `index.html` file and open it with the browser of your choice.
